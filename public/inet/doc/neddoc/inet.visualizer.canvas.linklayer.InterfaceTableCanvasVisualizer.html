InterfaceTableCanvasVisualizer

Package: inet.visualizer.canvas.linklayer

InterfaceTableCanvasVisualizer

simple module

This module visualizes interface tables on a 2D canvas. It displays a label for each selected network interface at the owner network node.

<b>See also:</b> ~InterfaceTableOsgVisualizer, ~InterfaceTableVisualizer, ~InterfaceTableVisualizerBase, ~IInterfaceTableVisualizer

InterfaceTableVisualizerBase simple module

This is a base module for interface table visualizer simple modules. It displays information of network interfaces at network nodes. Network(1,2,3) interfaces are not displayed by default, see module parameters for how to enable the visualization.
